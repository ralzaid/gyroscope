# Assignment 4- Gyroscope

## Overview

This project creates a simple tilt-based ball game using Jetpack Compose.

The ball moves based on the phone’s tilt using the accelerometer. The screen includes obstacles that the ball cannot pass through, creating a simple maze-like experience. The game is drawn using Canvas and uses a consistent pink color theme.

## AI Usage Disclosure

I used ChatGPT to help debug errors, simplify the code, and better understand how to use sensors in Jetpack Compose. It also helped me adjust the tilt and screenlock while testing.
